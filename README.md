# TanIA — Ferramentas com IA

Landing page da TanIA focada em uma suíte de ferramentas com IA para converter, organizar e validar conteúdo.

## Destaque

- Conversor de questões para XML Moodle
- Suporte a material bruto/desestruturado
- Identidade visual preto e branco
- Layout responsivo com foco em conversão

## Arquivos principais

- `index.html`
- `styles.css`
- `app.js`
- `assets/logo-tania-branca.svg`

## Observação

O link do primeiro projeto pode ser inserido no bloco "Ferramenta principal" quando estiver disponível.
